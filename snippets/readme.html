Instructions

Click on a vocabulary term to bring up all associated examples. Click on an example to load the associated media.

Console

Press enter after typing a command into the console to execute it.

x

Toggle simplified or traditional chinese.

p

Enable or disable the popup dictionary.

s

Enable or disable text selection for the popup dictionary.

r

Randomly shuffle or derandomize the vocabularly list.

e

Enable or disable randomly shuffling for examples.

(range)

Show terms in (range) from the vocabulary list. As a side effect, this operation will remove vocabulary randomization.
Example 1: "32" will show the 32nd term to the last term.
Example 2: "32-64" will show the 32nd term to the 64th term.

(term)

Inputs not matching any of the above will be treated as a vocabulary search. As a side effect, this operation will remove vocabulary randomization and ranges.
Example: "你" will search for all vocabulary containing "你".

Keyboard

Esc

Unselect vocabulary and examples, and show instructions.

Shift

Pause video if playing or play video if paused.

Left

Seek left 5 seconds (do nothing if 5 seconds left is before the start of the example).

Right

Seek right 5 seconds (do nothing if 5 seconds right is after the end of the example).

Hold Alt

Clicking on vocabulary or examples will not select them.

Preloading

When you select a vocabulary term, the first example will start preloading.
When you select an example, the example following that one will start preloading.
